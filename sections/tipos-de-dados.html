Tipos de dados

Tipos primitivos

Nome Tamanho Descrição
Boolean 1 bit 1 ou 0
Int8 1 byte Número inteiro de 8 bits com sinal
UInt8 1 byte Número inteiro de 8 bits sem sinal
Int16 2 bytes Número inteiro de 16 bits com sinal
UInt16 2 bytes Número inteiro de 16 bits sem sinal
Int32 4 bytes Número inteiro de 32 bits com sinal
UInt32 4 bytes Número inteiro de 32 bits sem sinal
Float32 4 bytes Número de ponto flutuante de 32 bits
Float64 8 bytes Número de ponto flutuante de 64 bits
DateTime 4 bytes Data e hora

Listas

Para declarar uma lista é necessário conhecer o seu tamanho máximo e o tipo de seus elementos.

Nome Tamanho mínimo Qtd. Elementos
Array8 1 bytes até 255 itens
Array16 2 bytes até 65.535 itens
Array32 4 bytes até 4.294.967.295 itens

Formato do buffer da lista:

Tamanho Elementos
1, 2 ou 4 bytes ELEMENTO 1 ELEMENTO 2 ... ELEMENTO N

Exemplo

    Tamanho do array: 4
    Array8<UInt16>[255, 512, 1024, 2048]
  
    0x04 0xFF 0x02 0x00 0x04 0x00 0x08 0x00

Buffers

Buffers são idênticos à Arrays, porém, são transformados em Buffer ou ArrayBuffer após a decodificação. Buffers podem ser do tipo Buffer8, Buffer16 e Buffer32 dependendo de seu tamanho assim como Arrays.

Strings

Strings seguem o mesmo princício dos Arrays, porém, não é necessário definir um tamanho máximo. Uma String acaba quando um caractere nulo 0x00 é encontrado.

Exemplo

    "Hello World!"
  
    72 101 108 108 111 32 87 111 114 108 100 33 0

JSON

JSON são idênticos à Strings, porém, são transformados em Objetos JavaScript utilizando a função JSON.parse() após a decodificação.
